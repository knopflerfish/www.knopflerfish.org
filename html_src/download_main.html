Knopflerfish 3.3.0 is the current production release. Download
Welcome to the download page for the Knopflerfish OSGi Service Platform. Below are links for downloading the latest production releases of KF3.x. There are also links for downloading the latest KF2 and KF1 maintenance releases. Older releases are kept in the archive section.

To get the latest development release of Knopflerfish either download a full distribution from the Nightly Builds section, or check out the source code and build it yourself.

Individual bundles can be downloaded from the Bundle repository

All bundles in the Knopflerfish releases are also available as maven artifacts in the Knopflerfish Maven repository.

All Knopflerfish releases are distributed under a BSD style license.
