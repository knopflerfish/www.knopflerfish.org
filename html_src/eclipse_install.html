Installation instructions

The Knopflerfish Eclipse Plugin is available under a BSD style license. You can download the latest release from the Knopflerfish update site, http://www.knopflerfish.org/eclipse-update/, using the Eclipse software update mechanism.

Note: if you are behind a proxy server, select 'Window->Preferences' then click 'Install/Update'. Check 'Enable HTTP proxy connection' and fill in the proxy server's host address and its port.

Using eclipse software update

Select 'Help->Software Updates->Find and Install...'.

Select 'Search for new features to install'.

Create a 'New Remote Site...'. Enter name, e.g. 'Knopflerfish update site', and URL 'http://www.knopflerfish.org/eclipse-update/'.

Check the box for Knopflerfish update site and press finish. If the update site is found you will see the features available on this site. Check the Knopflerfish OSGi IDE you wish to install.

Next you will be prompted with the license terms, which you must accept to continue. When you have accepted the license you can finish the installation.
