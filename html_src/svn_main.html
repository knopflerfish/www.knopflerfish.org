The Knopflerfish Source Code Repositories (subversion)

All code, resources and web pages for the Knopflerfish project are managed by subversion repositories. The repositories is open for reading to everyone.

Generic information on Subversion can be found at http://subversion.tigris.org

Knopflerfish.org repository - main KF source code repository

This is the main repository containg all the Knopflerfish OSGi source code. See

https://www.knopflerfish.org/svn/knopflerfish.org/readme.txt

for details on how to check out the repository.

The entire repository is browsable from

https://www.knopflerfish.org/svn

Knopflerfish Eclispe Plugin repository

Due to historical reasons the Knopflerfish Eclipse Plugin is maintained in a seperate repository,

See

https://www.knopflerfish.org/svn/eclipse.knopflerfish.org/readme.txt

for details on how to check out the Knopflerfish Eclipse Plugin repository.

This entire repository is browsable from

https://www.knopflerfish.org/svn/eclipse.knopflerfish.org/
