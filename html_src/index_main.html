Knopflerfish - Open Source OSGi

Welcome to the home of Knopflerfish, the leading universal open source OSGi Service Platform. Led and maintained by Makewave, Knopflerfish delivers significant value as the key container technology for many Java based projects and products.

Knopflerfish 3.2 is released!

The second minor release of Knopflerfish 3 is now available. KF3 is designed in accordance with the OSGi R4 v4.2 specifications and is a full implementation of the OSGi framework as well as the OSGi Compendium Services. The Release Notes includes the complete list of new features in KF 3.2.0

Download Knopflerfish - OSGi R4 v4.2, R4 & R3

Knopflerfish 3.x is the current main version of Knopflerfish. It is designed in accordance with OSGi Release 4, version 4.2 specification.

KF 2.x (OSGi R4) and KF1.x (OSGi R3) remain as OSGi Release 4 and Release 3 options, both available for download

Knopflerfish Pro - certified and supported
