Knopflerfish - Open Source OSGi SDK and run-time container

Welcome to the home of Knopflerfish, the leading universal open source OSGi Service Platform. Led and maintained by Makewave, Knopflerfish delivers significant value as the key container technology for many Java based projects and products.

Knopflerfish 5 is released! - OSGi R5

Knopflerfish 5, the new major release of the leading open source OSGi run-time for the embedded market, is now available for download. Knopflerfish 5 is designed to be implemented according to the OSGi R5 specifications. The Release Notes includes the complete list of new features in KF 5.

Knopflerfish 4 and 3 - stable OSGi R4 platforms

With the release of Knopflerfish 5, KF4 moves to maintenance mode alongside with KF3. Both KF4 and KF3 will remains as stable OSGi R4 v4.3 and v4.2 alternatives for a foreseeable future. Anyone preferring OSGi R4 over R5 can consider them as full-worthy production platforms. The latest version of current or former production releases are always available from the Download section.

Download Knopflerfish - OSGi R5 to R3

Knopflerfish 5.x is the current production release of Knopflerfish. It is designed in accordance with OSGi Release 5.

KF 4.x (OSGi R4 v4.3), KF 3.x (OSGi R4 v4.2), 2.x (OSGi R4), and KF1.x (OSGi R3) remain as OSGi Release 4 and Release 3 options available for download
