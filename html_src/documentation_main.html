Knopflerfish Manuals and Documentation

The Knopflerfish manuals, tutorials and API documentation are all part of the Knopferfish distribution. The manuals and API documentation for the latest KF release is always available on-line from here. The manual for the latest development version is also available in the left menu, via the Manuals (trunk) link.

Documentation for older releases also available on-line from the respective releases directory, accessible via the Downloads section.

Overview of the Knopflerfish OSGi distributions

There have been three major releases of Knopflerfish. The current major release is Knopflerfish 3. KF3 is designed in accordance with the OSGI R4 v4.2 specifications.

More detailed information on the contents on each of the major KF versions can be found under each of the KF3, KF2 and KF1 links below.
