The Knopflerfish Source Code Repositories (Git)

All code, resources and web pages for the Knopflerfish project are managed by github repositories. The repositories is open for reading to everyone.

Generic information on GitHub can be found at http://github.com

Knopflerfish.org repository - main KF source code repository

This is the main repository containg all the Knopflerfish OSGi source code. See:

https://github.com/knopflerfish/knopflerfish.org

Knopflerfish Eclispe Plugin repository

Due to historical reasons the Knopflerfish Eclipse Plugin is maintained in a seperate repository, See:

https://github.com/knopflerfish/eclipse.knopflerfish.org
