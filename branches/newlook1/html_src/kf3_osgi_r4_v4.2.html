About Knopflerfish 3 - an OSGi R4 v4.2 distribution

KF3 is the latest major release of Knopflerfish. KF3 is an implementation of the OSGi R4 V4.2 specifications. The distribution includes the following components:
  • Knopflerfish OSGi framework - OSGi R4 v 4.2 Core Specification implementation
  • Knopflerfish OSGi bundles - OSGi R4 v 4.2 Service Compendium implementation
  • Knopflerfish components & extras - Knopflerfish bundles and utilities

Knopflerfish 3 Features and changes compared to KF2

The Knopflerfish 3 OSGi framework OSGi Compendium Services are all implemented in accordance with the OSGi R4 v4.2 specifications.

The desktop and framework commands have been updated in a number of ways to reflect new features in R4 V4.2

The table below provides an overview of the components included as well as comments on features specific for OSGi R4 v4.2 where applicable. The numbering for the OSGi defined components refer to the OSGi specification chapter numbering.
