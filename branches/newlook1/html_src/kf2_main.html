Knopflerfish 2, OSGi R4

Knopflerfish 2, Major Release of KF

This is work in progress on KF2, a major release of KF. The objective is to implement the OSGi R4 specifications fully. This includes the:
  • Core Specification
  • Service Compendium
  • Mobile Specification, MEG - Not released yet.
A first version has been released and is available for download.

Status KF 2.0.0 release

With this first release, the KF 2 framework is feature complete in all mandatory parts. Of the optional parts only some security items remain. All other parts are implemented and fully functional.

The desktop and framework commands have been updated in a number of ways to reflect new features in R4.

The table below describes what parts have been implemented and what parts are still pending in the KF 2.0.0 release.
