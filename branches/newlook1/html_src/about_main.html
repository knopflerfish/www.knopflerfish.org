About Knopflerfish

Makewave (formerly Gatespace Telematics) is the primary maintainer and sponsor of the Knopflerfish project and has several developers assigned to develop and maintain the Knopflerfish OSGi distribution.

The OSGi Alliance specifies an industry standard Java application platform, allowing multiple applications, to securely run in a single JVM. These programs can share resources as data, functionality and threads. Further information on the OSGi technology is available at the OSGi Alliance web site at: http://www.osgi.org
